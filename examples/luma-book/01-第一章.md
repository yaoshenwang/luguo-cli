# 斜率是什么

一次函数写作 $y = kx + b$,其中 $k$ 是斜率。

:::quiz 斜率为负代表什么?
- [ ] 直线水平
- [x] 直线从左上向右下倾斜
- [ ] 直线经过原点
@id q-ch1-slope-sign
@explain k < 0 时 x 增大 y 减小,直线下降。
@skills 判断斜率方向
@steps 读取 k 的符号,判断 y 的变化,匹配直线趋势
:::

:::keypoints 核心概念
- **斜率 k**: y 的变化量除以 x 的变化量
@skills 计算两点斜率
:::

:::quiz 过点 (0, 1) 与 (2, 5) 的直线斜率是多少?
- [ ] 1
- [x] 2
- [ ] 4
@id q-ch1-slope-points
@explain 斜率等于 (5-1)/(2-0)=2。
@skills 计算两点斜率
@steps 计算 y 变化量,计算 x 变化量,两者相除
:::

:::quiz k = 0 的直线是什么方向?
- [x] 水平
- [ ] 竖直
- [ ] 必过原点
@id q-ch1-slope-zero
@explain k=0 时 y 不随 x 变化,所以直线水平。
@skills 识别零斜率图像
@steps 代入 k=0,判断 y 是否变化,匹配图像方向
:::
