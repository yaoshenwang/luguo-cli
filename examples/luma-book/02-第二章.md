# 截距与图像

截距 $b$ 是直线与 y 轴交点的纵坐标。

:::keypoints 核心概念
- **纵截距 b**: 当 x=0 时的 y 值。
:::

:::example 求截距
直线过点 (0, 3),截距是多少?
1. x = 0 时 y = 3
@answer b = 3
:::

:::quiz y = 2x + 5 的截距是?
- [ ] 2
- [x] 5
- [ ] 7
@id q-ch2-intercept
@explain x = 0 时 y = 5。
@skills 纵截距
@steps 令 x=0,计算 y,读取纵截距
:::

:::quiz 直线 y = -3x + 2 与 y 轴交于哪里?
- [ ] (2, 0)
- [x] (0, 2)
- [ ] (0, -3)
@id q-ch2-axis-crossing
@explain y 轴上的点满足 x=0,代入得到 y=2。
@skills 纵截距
@steps 识别 y 轴条件,代入 x=0,写出交点
:::

:::quiz 哪个改动只会改变截距而不改变斜率?
- [x] y=2x+1 改成 y=2x+4

- [ ] y=2x+1 改成 y=-2x+1
@id q-ch2-shift
@explain 斜率系数仍为2,只有常数项 b 改变。
@skills 纵截距
@steps 比较 x 系数,比较常数项,判断图像平移
:::
